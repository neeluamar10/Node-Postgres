# Node-Postgres

1. Install dependencies
2. run index.js file
3. It is hosted in localhost:3000
4. you can navigate to /users -- to look for the users present in the database
5. you can navigate to /users/2 -- to retrieve the user with id 2
6. you can perform createUser, updateUser, and deleteUser using the Thunder Client in Visual studio code
7. Install Thunder Client
8. make a New Request and go to the "body" tab and select appropriate CRUD operation which you would like to perfom
9. give the information in json format.
10. you could see the result in the right side response window.
